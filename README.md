# 🛢️ Oils & Fats Market Terminal

AI-powered analytics terminal for vegetable oils & fats market. Real-time prices, news, and AI-generated market intelligence.

![Terminal Preview](https://i.imgur.com/placeholder.png)

## ✨ Features

- **📊 Live Market Data** — Real-time prices from Google Search (Serper API)
- **📰 News Aggregation** — Latest market news from Tavily API
- **🤖 AI Analysis** — Market reports generated by Gemini/Groq
- **📄 PDF Export** — Download reports as PDF
- **🔗 Share** — Share reports via Web Share API or clipboard
- **🔄 Auto-refresh** — Data updates every 15 minutes

## 🚀 Quick Start

### 1. Clone & Install

```bash
git clone https://github.com/YOUR_USERNAME/oils-fats-terminal.git
cd oils-fats-terminal
npm install
```

### 2. Configure Environment Variables

```bash
cp .env.example .env
```

Edit `.env` and add your API keys:

```env
SERPER_API_KEY=your_key_here
TAVILY_API_KEY=your_key_here
GEMINI_API_KEY=your_key_here
VITE_GROQ_API_KEY=your_key_here
```

**Get free API keys:**
- [Serper](https://serper.dev) — 100 free searches
- [Tavily](https://tavily.com) — 1,000 free searches/month
- [Gemini](https://makersuite.google.com/app/apikey) — Free tier
- [Groq](https://console.groq.com) — Free tier available

### 3. Run Locally

```bash
npm run dev
```

Open http://localhost:5173

### 4. Deploy to Vercel

```bash
npm i -g vercel
vercel
```

**Set environment variables in Vercel Dashboard:**
1. Go to Project Settings → Environment Variables
2. Add: `SERPER_API_KEY`, `TAVILY_API_KEY`, `GEMINI_API_KEY`

## 🔌 API Endpoints

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/api/prices` | GET | Live vegetable oil prices |
| `/api/news` | GET | Latest market news |
| `/api/ai-analysis` | POST | Generate AI report |
